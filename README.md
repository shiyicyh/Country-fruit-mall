# Country-fruit-mall
“果几来啦”水果商城小程序版

### 系统介绍：

“果几来啦”商城主要分为两大部分：面向普通用户的微信小程序客户端和面向运营商 

的网络后台管理端。客户端主要实现产品浏览、产品搜索、添加购物车、下单、查看订单、 

收货评价等功能；管理端主要实现产品管理、评论管理、订单管理、分类管理等功能。
